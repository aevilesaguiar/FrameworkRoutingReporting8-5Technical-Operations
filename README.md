# Framework Routing & Reporting 8.5 Technical Operations

This course is intended for anyone to learn the basic functionality of Genesys Framework Routing & Reporting 8.5 Technical Operations.   

Note: Genesys Engage Cloud (formerly known as PureEngage Cloud) is in the process of transitioning to Genesys Multicloud CX. The change is in name only, and all of our Genesys Engage Cloud training supports Genesys Multicloud CX.

Este curso destina-se a qualquer pessoa que aprenda a funcionalidade básica do Genesys Framework Routing & Reporting 8.5 Technical Operations.

Observação: o Genesys Engage Cloud (anteriormente conhecido como PureEngage Cloud) está em processo de transição para o Genesys Multicloud CX. A mudança é apenas nominal e todos os nossos treinamentos Genesys Engage Cloud são compatíveis com o Genesys Multicloud CX.
